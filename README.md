# 🎓 WordPress pour les Noobs

Documentation complète pour apprendre WordPress étape par étape.

## 📚 Contenu

Cette documentation couvre :

- ✅ Introduction à WordPress
- ✅ Installation avec Local WP
- ✅ Familiarisation avec l'interface
- ✅ Gestion du contenu (articles, pages)
- ✅ Médiathèque et optimisation
- ✅ Éditeur de blocs Gutenberg
- ✅ Configuration et réglages
- 🔄 Et bien plus à venir...

## 🚀 Fonctionnalités du site

- 📱 **Responsive** : Optimisé mobile/desktop
- 🔍 **Zoom d'images** : Cliquez sur les captures d'écran
- 🧭 **Navigation fluide** : Table des matières interactive
- ⚡ **Performance** : Chargement rapide
- 🎨 **Interface moderne** : Design professionnel

## 🛠️ Technologies utilisées

- **Frontend** : HTML5, CSS3, JavaScript
- **Markdown** : Marked.js pour le parsing
- **Syntaxe** : Highlight.js pour la coloration
- **Hébergement** : GitHub Pages
- **Responsive** : CSS Grid + Flexbox

## 📖 Utilisation locale

```bash
# Cloner le repository
git clone https://github.com/ChapeauDent/WordPress_pour_les_nuls.git

# Ouvrir le fichier index.html dans votre navigateur
# Ou servir via un serveur local (recommandé)
npx http-server .
```
